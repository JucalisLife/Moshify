## About the Project
Moshify is the landing page of a fictional cloud hosting service built using HTML and CSS with fully implemented responsiveness across multipule screens.

## Built with

- HTML
- CSS
- JavaScript

## Usage
Follow the following steps in order to run Moshify locally:

1. Clone the repository:
`https://github.com/JucalisLife/Moshify.git`
2. Navigate to the project directory:
`cd Moshify`
3. Open index.html in your preferred web browser.
